# CroquisChallenge
## 🌵 개요

* 꾸준한 크로키 연습을 통해 실력을 향상시키고 싶은 이들을 위한 App 
* 다른 사람들의 작품을 감상하고 나의 작품을 다른 사람들과 공유하고 싶은 이들을 위한 App


## 🖥️ 개발 환경
* Android studio 4.1
* Firebase
* Adobe XD
